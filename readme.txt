The easiest way to embed YOGO Booking on your Wordpress website.

== Description ==
This plugin enables [YOGO](https://yogobooking.com) integration on your Wordpress website.

[Official documentation](https://docs.yogo.dk/da/articles/6620472-integration-med-wordpress)


== Installation ==
1. Activate the plugin.
2. Go to the YOGO Booking settings page and enter your YOGO domain. It will look something like `mystudio.yogobooking.com`.
3. Try typing `[yogo-calendar]` on one of your pages. It should show the schedule for your studio.
4. If nothing happens, check that you entered your domain correctly.
5. You're done!

== Frequently Asked Questions ==
**Is this a booking system?**
[YOGO](https://yogobooking.com) is a booking system. This plugin makes it easy to embed your schedule, price list, events, teacher list and more into your Wordpress website.

**Do I need an account with YOGO to use this plugin?**
The plugin itself is free and open source, but you will not have any use for it without a YOGO account.
